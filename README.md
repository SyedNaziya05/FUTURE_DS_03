# FUTURE_DS_03: Marketing Funnel Analysis

## Objective
To analyze the effectiveness of the Bank Marketing campaign by tracking customer conversion from initial contact to term deposit subscription.

## Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

## Key Metrics
* **Total Contacts:** 45,211
* **Successful Subscriptions:** 5,289
* **Conversion Rate:** 11.7%

## Key Insights
* The campaign reached 45,211 prospective clients.
* An 11.7% conversion rate indicates that while the campaign reached many, a more targeted approach is needed to improve efficiency.

## Recommendations
* Focus future marketing efforts on demographic segments that showed a higher propensity to subscribe.
* Optimize call timing and frequency to improve engagement.
